simple Blackjack

Animated card game

simpleBlackjack is a browser based blackjack game written in HTML, CSS, and JS. It allows for hitting, standing, doubling, and splitting. All phases of the game are animated, including card dealing, card flipping, card shuffling, and a selection effect causing the card to hover. The animations make heavy use of Promises to ensure the timing of the animations is perfect.

This is the third iteration of simpleBlackjack. The first was a terminal game written in Python; the second was a browser based game written in HTML, CSS, and JS, but it effectively looked like a terminal game. The final iteration added complex animations to the core of the second iteration.

Go ahead and give it a try by clicking any of the images or the hyperlink.
